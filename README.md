# AndriiNeverov Tap

## How do I install these formulae?

`brew install andriineverov/tap/<formula>`

Or `brew tap andriineverov/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "andriineverov/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
